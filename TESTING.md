# Testing & Reports

Run everything:

```bash
npm run ci
```

Run backend tests:

```bash
npm run test:backend
```

Run frontend tests:

```bash
npm run test:frontend
```

Generate coverage and JUnit reports on demand:

```bash
npm run test:report
```

Report outputs:

- `Backend/coverage`
- `Backend/reports/junit/backend-junit.xml`
- `Frontend/coverage`
- `Frontend/reports/junit/frontend-junit.xml`
